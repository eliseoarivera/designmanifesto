DESIGN MANIFESTO

My thoughts web design.

Font

Fonts should be clean. I like to use Sans Serif fonts in general. A webpage should only have one or two fonts utilized for the whole page. This page uses Archivo Black for all headings and Montserrat for the cards' body text.

Color

Use color sparingly. I like to use monochrome whenever I can. Color schemes that work well reflect logos or graphics in the webpage and/or add empahsis to or connect similar content.

Spacing

There should be enough space to tell which blocks of text belong with what. Use padding to create space and provide breathing room between content. For example, I have added more or less space between the heading and cards.

Alignment

You should use alignment to help orient the user and give them an organized space. The cards' text has been aligned to the left while the heading keeps a center alignment. The cards are also aligned using a 2x2 grid.
